# AI Studio - 智能创作平台

基于免费在线API和本地AI的图像和视频生成平台。 

## 功能特性

- 🎨 **AI图像生成** - 使用文本描述生成精美图像
- 🎬 **AI视频生成** - 将创意转化为动态视频
- 📁 **历史记录** - 管理和查看历史作品
- ⚙️ **系统设置** - 灵活配置生成模式

## 双模式设计

### ☁️ 在线模式（推荐）
- **免费服务**：使用 Pollinations AI，完全免费
- **无需配置**：无需注册账号，即开即用
- **快速生成**：服务器端推理，生成速度快
- **功能完整**：支持图像和视频生成

### 🖥️ 本地模式（隐私优先）
- **完全隐私**：所有推理在本地浏览器完成
- **离线可用**：首次下载后可离线使用
- **无需联网**：数据不离开您的设备
- **技术栈**：Transformers.js + WebGPU

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **编程语言**: TypeScript
- **样式方案**: Tailwind CSS
- **状态管理**: Zustand
- **本地AI**: Transformers.js (WebGPU)
